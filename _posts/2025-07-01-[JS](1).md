### JS로 HTML 조작하는 법
- html 문서 내 `<script>` 태그에 작성
- 바꿀요소id / 무엇을 / 어떻게
- 셀렉터: id 등 이용해 html 요소 찾는데 활용 
    ```js
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>

        <h1 id="hello">안녕하세요</h1>

        <script>
            document.getElementById('hello').innerHTML = '안녕';
            document.getElementById('hello').style.color = 'blue';
            document.getElementById('hello').style.fontSize = '16px';
        </script>

    </body>
    </html>
    ```

## 동적 UI 만드는 스텝 (Alert 박스 만들기) 
---
### UI 만드는 step
- 케이스 스터디 식으로 배울 수 있으나 직접 만드는 훈련
0. 무수한 UI를 위해
1. HTML/CSS로 미리 디자인
2. 필요할 때 해당 UI 띄우는 코드를 자바스크립트로 짬

### Alert 박스 만들기 step 1
- HTML/CSS로 미리 디자인
- `index.html`
    ```html
    <body>
    <div class="alert-box">알림창</div>
    </body>
    ```
- `main.css`
    - 숨기고 싶다면 `display: none;`
    ```css
    .alert-box {
        background-color: skyblue;
        padding: 20px;
        color: white;
        border-radius: 5px;
        display: none;
    }
    ```

### Alert 박스 만들기 step 2
- 버튼 누를 시 UI를 보여달라고 코드 짜기
- `main.css`
    ```js
    <button onclick="document.getElementById('alert').style.display = 'block';">버튼</button>
    ```
